# Teslo API

1. Clonar el proyecto
2. Instalar dependencias ```yarn install```
3. Clonar el archvio `.env.template` y renombrarlo a `.env`
4. Cambiar las variables de entorno
5. Levantar la base de datos

```bash
docker compose up -d
```
